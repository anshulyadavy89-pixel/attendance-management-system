# Attendance Management System

A simple Attendance Management System built using Python and SQLite.

## Features

* Add Student
* View Students
* Mark Attendance
* View Attendance Records

## Technologies Used

* Python
* SQLite

## How to Run

1. Run database.py
2. Run main.py
3. Add students
4. Mark attendance
5. View attendance records
